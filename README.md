# 📊 Customer Order Management System (Excel-Based Project)

This project simulates a **Customer Order Management System** using Microsoft Excel, designed to demonstrate data entry, processing, and organization for real-world business operations. It's ideal for practicing key data handling tasks such as tracking orders, calculating totals, and managing payment statuses.

## 🔍 Project Overview

The dataset consists of 300 records representing customer orders from a fictional e-commerce system. It includes customer names, email IDs, contact numbers, product details, quantities, pricing, and payment statuses.

This Excel workbook is structured to help understand how small businesses manage order data manually before scaling to more advanced tools like databases or CRMs.

## 🧩 Features

- ✅ Clean and structured order data entry system  
- ✅ Customer names and email addresses  
- ✅ Order IDs  
- ✅ Formulas to calculate total order price  
- ✅ Conditional formatting to highlight unpaid orders  
- ✅ Filterable columns for quick search and sort  

## 🛠️ Tools Used

- **Microsoft Excel**
- **Data validation**, **conditional formatting**, and **formulas**
